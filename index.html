Quem Sou Eu?

Sou desenvolvedor full stack com experiência no desenvolvimento de aplicações web utilizando HTML, CSS, JavaScript e ReactJS no front-end, sempre buscando criar interfaces responsivas, acessíveis e com boa usabilidade. No back-end, atuo com PHP, integrando sistemas com banco de dados, criando APIs e desenvolvendo soluções personalizadas de forma segura e eficiente. Além do desenvolvimento web, possuo conhecimentos sólidos em Linux, o que me permite trabalhar com servidores, automações de tarefas, deploys, configuração de ambientes e análise de logs. Essa base sólida em sistemas Linux complementa minha atuação como desenvolvedor, garantindo maior autonomia e entendimento completo da infraestrutura por trás das aplicações que construo.
